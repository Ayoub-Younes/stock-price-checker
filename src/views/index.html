Nasdaq Stock Price Checker

How It Works

This tool allows you to retrieve stock prices for any Nasdaq-listed company by entering the stock symbol. You can also add a like to a stock.
Note: Only one like is allowed per IP address.

For a full list of Nasdaq stock symbols, visit the link below:

Nasdaq Stock List

Example API Usage

Use the following API endpoints to get stock data:

  • Get data for GOOG
  • Get data for GOOG and like
  • Compare GOOG and MSFT
  • Compare GOOG and MSFT with like

Example API Responses

          Single Stock: {"stockData":{"stock":"GOOG","price":786.90,"likes":1}}
        
          Comparison: {"stockData":[{"stock":"MSFT","price":62.30,"rel_likes":-1},{"stock":"GOOG","price":786.90,"rel_likes":1}]}
